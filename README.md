# team-task-manager-uni
チームで取り組む大学のレポートや課題を管理し、進捗を可視化するためのPythonアプリ

## 1. 開発の背景
大学のグループワーク部活動の役職において、「誰がどのタスクを担当しているか」「現在の進捗状況はどうなっているか」が不明確になり、提出直前に混乱するケースが多く見られます。
この課題を解決するため、チームメンバー全員がリアルタイムでタスク状況を把握・更新できる管理ツールを開発します。

## 2. 実装機能 (CRUD)
本アプリでは、以下の主要機能を実装予定です。

- **タスク作成 (Create)**: 課題名、担当者、期限、詳細内容の登録。
- **タスク表示 (Read)**: チーム全体のタスク一覧、および個人の担当タスク表示。
- **タスク更新 (Update)**: 進捗状況（未着手 / 進行中 / 完了）の変更。
- **タスク削除 (Delete)**: 完了したタスクや誤登録したタスクの削除。
